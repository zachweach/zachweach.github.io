CSCI0111: Computing Foundations: Data
link to the course webpage
for fall 2021 and spring 2022
an introductory computer science course at brown. it teaches the basics of computer science and coding, including data structures (especially tabular data) and algorithms, with a focus on applications in data science. it is primarily taught in python and pyret, a language developed for computer science education.
in this position, i held weekly 1-on-1 office hours to assist students with the conceptual content of the course and debugging. i provided the same help asynchronously via a course-wide discussion board. additionally, i held weekly "lab sections", where students would apply the skills they have learned and write code in a low-pressure learning environment.
in fall 2020, i served on the "projects team", where I was responsible for creating and maintaining project handouts, solution code (written in pyret), resources/assets, and stencil code for students.
CLPS0900: Statistical Methods
