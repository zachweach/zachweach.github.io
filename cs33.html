atoi

written in x86_64 assemply, this project implements the built-in C function atoi(), which converts a string argument to an integer.

design & implementation

atoi() goes through each ascii character in the input and loops through each one until it either reaches the end or finds a non-numerical character.

one of the biggest challenges of this project was that it had to be written entirely in machine code and not some other higher-level langauge, so i had to manage registers and accumulators in the system processor.

honestly, this project was not very long, but gave me a really good understanding of how registers and low-level processing work.
